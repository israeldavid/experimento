# Proyecto Prueba de Implementación

Este proyecto sirve para ir probando los cambios de arquitectura de la aplicación.

## Comenzando 🚀

_El Objetivo es descargar el proyecto e ir probando los servicios y componentes en la aplicación de peliculas e ir implementando los métodos que cumplan con una arquitectura limpia._

Mira **Deployment** para conocer como desplegar el proyecto.


### Pre-requisitos 📋

_Necesitas Node.js, ionic 5, angular para poder ir cambiando y probando las nuevas funciones_

```
npm install -g ionic
```
